# Manual vs AI

**Source:** https://www.getcairn.dev/docs/manual-vs-ai
**Captured:** 2026-04-28

## Overview

Cairn is an AI-native model-based systems engineering platform that transforms rough system descriptions into structured, persistent engineering models. Unlike typical AI chat tools that produce disposable transcripts, Cairn maintains an inspectable, editable model that survives beyond the conversation.

## Key Features

### Persistent Models
"Chat vanishes. The model stays." While generic AI tools provide transcripts that expire, Cairn keeps structured engineering models with explicit nodes, requirements, and interfaces that users own and edit.

### AI-Guided Decomposition
The platform uses an AI interview process to surface subsystems, components, and interfaces from natural language descriptions. Users provide rough outlines, and the system synthesizes structure through clarifications and inferences.

### Multiple Analytical Views
Twelve distinct lenses allow inspection of the same underlying model from different angles:
- Overview, Brief, Visuals, Requirements, Architecture
- Causality, Completeness, Narrative, Dendritic
- Behavior, Verification, Operational

### Integrated Depth Tools
- 2D concept renders
- 3D mesh assets
- Monte Carlo simulation
- All artifacts tied to the model for full traceability

### Reviewable Change Management
Proposed structural changes arrive as ChangeSets that users can accept, skip, or reject individually. The model updates instantly upon acceptance without out-of-band edits.

### Complete Traceability
Every requirement connects to implementing components, verification methods, and results. Requirements remain queryable and current because they're part of the live model.

## Core Philosophy

"The model is the artifact, not the conversation." Cairn preserves system trees, requirements, interfaces, verifications, and change history while eliminating disposable chat transcripts and context-dependent scaffolding.

## Getting Started

Users describe systems in plain English. Rough descriptions are acceptable. Cairn refines them through the Describe, Decompose, Inspect, Refine cycle. No account required to try the platform.
