## Summary
This panel presents a single-system fidelity assessment for "Launch and Recovery System" using a three-axis radar (ternary) chart with axes labelled Entities, Processes, and Relationships. The plotted polygon is collapsed: Entities at 0 percent, Processes at 0 percent, and Relationships at 100 percent, producing a degenerate line rather than a filled triangle. Three metric tiles below the chart restate each axis with sub-detail (children defined, behaviors defined, interfaces and reqs counts). A warning callout flags structural-but-not-behavioral coverage and offers a "Fix with AI" remediation action.

- Color coding is consistent across chart axes, tile top borders, and vertex dots: ochre = Entities, teal/green = Processes, purple = Relationships. Portable as a tri-color taxonomy convention.
