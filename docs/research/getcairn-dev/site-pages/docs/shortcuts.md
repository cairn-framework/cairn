# Keyboard Shortcuts

**Source:** https://www.getcairn.dev/docs/shortcuts
**Captured:** 2026-04-28

| Shortcut | Action | Context |
|----------|--------|---------|
| Command+K | Open command palette | Global |
| Esc | Close panel / modal | Global |
| ? | This help | Global |
| 1 to 7 | Switch lenses (Overview through Operational) | Workspace |
| Command+Shift+Q | Toggle Quality tool | Workspace |
| Command+Shift+H | Toggle History tool | Workspace |
| Command+Shift+T | Toggle Traceability tool | Workspace |
| Delete / Backspace | Delete selected node (Architecture) | Workspace |
| Command+Enter | Save in popover editors | Inspector |
