# cairn domain expandability

Analysis of whether cairn extends to non-software domains (hardware MBSE, org-structure, research programmes). See archive/strongholds/cairn-domain-expandability.md for full analysis.

Key conclusions: the blueprint grammar (System/Container/Module/Actor) and the two-chain model are domain-agnostic. The code reconciler is the only domain-specific component; the reconciler interface is designed to be pluggable (V2-04). Non-code reconcilers are explicitly deferred post-v1.
